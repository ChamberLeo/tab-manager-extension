TAB MANAGER

Privacy Policy - Last Modified: February 3, 2026

Summary

Tab Group Manager is committed to protecting your privacy. This extension operates entirely offline and does not collect, transmit, or share any user data.

Data Collection

We do NOT collect any personal or sensitive user data, including but not limited to:

  • Personally identifiable information
  • Browsing history
  • Website content
  • Cookies or cached items
  • Form data
  • Financial and payment information
  • Health information
  • Authentication information
  • Communication content

Data Transmission

We do NOT transmit any personal or sensitive data via secure or insecure networks. No user data is shared with third parties.

Analytics and Tracking

This extension does NOT use any analytics services or tracking mechanisms. We do not collect usage statistics, error reports, or any other telemetry data.

Local Storage

The extension uses Chrome's local storage API to store:

  • Saved tab groups - Names, colors, and URLs you choose to save
  • Theme preference - Your light/dark mode setting

This data is stored only on your device and is never transmitted externally. You can delete this data at any time by uninstalling the extension or clearing your browser data.

Permissions

The extension requires the following permissions for core functionality:

Permission Purpose
tabs Read tab information (title, URL, icon) for display and organization
tabGroups Create and manage Chrome's native tab groups
storage Save your preferences and tab groups locally

These permissions are used solely for the extension's functionality and not for data collection.

Third-Party Services

The extension loads the "VT323" font from Google Fonts for visual styling. This is the only external connection made by the extension. No personal data is sent through this connection. For more information, see Google's Privacy Policy.

Export and Import

  • Export creates a local JSON file on your device
  • Import reads local files from your device

No data is transmitted over the internet during these operations.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last Modified" date above.
